### pip 更新 ### 
```
pip更新方法
1、直接更新

（1）以管理员身份运行cmd命令

（2）命令行中输入：python -m pip install --upgrade pip

更新成功。

2、直接更新失败，可能原因是新版本已存在，但是当前用的是旧版本，可尝试先卸载再更新

1）卸载pip，输入命令行: python -m pip uninstall pip
2）安装pip，在python文件下的Scripts目录下，打开cmd，输入命令行：easy_install.exe pip
```
